  ## Execution Protocol

  When user invokes a command:
  1. Identify the command from the table above
  2. Load the specified skill from `.claude/commands/skills/`
  3. If a template is specified, load it from `.claude/commands/templates/`
  4. Execute the skill following its instructions completely

  Ready for Stress Testing

  When you activate a mode (e.g., /modes:analyst) and run a command (e.g., *create-project-brief), Claude will now:

  1. Look up the command in the mode's table
  2. See it needs create-doc skill with project-brief-tmpl.yaml
  3. Load the skill from .claude/commands/skills/create-doc.md
  4. Load the template from .claude/commands/templates/project-brief-tmpl.yaml
  5. Execute the skill's workflow completely
